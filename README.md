# SNU 뉴스 조직이해 자료집

서울대학교 [조직이해] 자기소개서 문항 답변 준비를 위한 자료 수집 및 PDF 생성 프로젝트.

## 지원 문항

> 지원자가 생각하는 서울대학교의 핵심가치는 무엇인지 기술하고, 최근 대학을 둘러싼 다양한 이슈 중 하나를 선택하여 이에 대한 지원자의 견해와 서울대학교의 역할에 대하여 기술하여 주십시오.

## 핵심 발견 사항 (자소서 논거)

### 서울대 핵심가치 (공식 자료 기반)
- **공식 비전**: "대전환 시대를 이끌어가는 학문공동체"
- **80주년 표어**: "무한한 상상, 담대한 도전, 새로운 미래"
- **총장 철학**: "대학의 생명은 자유" (유홍림 총장)
- **4대 전략**: 융합 인재 양성 / 인류 과제 해결 / 운영 혁신·자율성 / 지속가능성

### 2026년 최대 이슈 후보 3개

1. **민주주의 위기 (탄핵 파면 1주년)**
   - 2024.12.03 비상계엄 → 2025.04.04 헌재 만장일치 파면
   - IFS 연구: 한국인 20~25%가 권위주의 대안 수용 의향
   - 서울대 역할: 공론장·시민교육 허브

2. **AI 전환과 대학 재편**
   - AI 전문대학원 2027년 개원 목표
   - TransLive(실시간 번역), 융합AI광역(100명), Grand Quest
   - 데이터 주권·AI 윤리 거버넌스 과제
   - 서울대 역할: AI 교육·연구·윤리의 선도자

3. **학령인구 절벽 + 고등교육 불평등**
   - 2026년 학령인구 483만 명 (500만 붕괴)
   - 사립대구조개선법 2026.08 시행, 대학 폐교 명령 가능
   - 수도권 충원율 99% vs 비수도권 83%
   - 서울대 역할: "서울대 10개 만들기" 지지, 생태계 책임
