# CLAUDE.md — 컨텍스트 유지 파일

Claude Code가 이 프로젝트를 다시 열 때 참고할 작업 맥락 및 지침.

---

## 프로젝트 목적

서울대학교 직원 채용 자기소개서 [조직이해] 문항 준비.

**문항**: 지원자가 생각하는 서울대학교의 핵심가치는 무엇인지 기술하고, 최근 대학을 둘러싼 다양한 이슈 중 하나를 선택하여 이에 대한 지원자의 견해와 서울대학교의 역할에 대하여 기술하여 주십시오.

---

- **완료**: 3개 출처에서 기사 53편 수집 → PDF v3 생성 → GitHub 업로드

- **다음 단계**: 이슈 선택 후 자소서 초안 작성

## 핵심 파일

| 파일 | 용도 |
|---|---|
| `SNU_뉴스_종합_자료_v3.pdf` | 최종 자료집 (53편, 7섹션) — 열람용 |
| `create_pdf_v3.py` | PDF 재생성 스크립트 |

| `snu_full_report.txt` | 수집 기사 58편 텍스트 보고서 |

## 자소서 이슈 후보 3개 (미결정)

사용자가 아직 어떤 이슈로 쓸지 결정하지 않았음.

1. **민주주의 위기 (탄핵 파면 1주년)** — 임팩트 강함
2. **AI 전환과 대학 재편** — 서울대 전략 방향과 직결
3. **학령인구 절벽 + 고등교육 불평등** — 구조적 큰 그림

## 서울대 핵심가치 공식 자료 요약

- 비전: "대전환 시대를 이끌어가는 학문공동체"
- 80주년 표어: "무한한 상상, 담대한 도전, 새로운 미래"
- 총장 철학: "대학의 생명은 자유" (유홍림 총장, 2026 신년사)
